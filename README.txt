Ce jeu a été développé dans le cadre des projets PPD de seconde année de l'ESIEA.
Le but de ce jeu est de faire découvrir l'école de façon ludique aux visiteurs en leur proposant de jouer lors des journées portes ouvertes.

Equipe 2012-2013: V. Marquet, E. Baudvin, Q. Bienvenu, P. Berettoni

Technical:
This game can be played on the Wii or with the emulator Dolphin.
It uses the GRRLIB graphic library, designed specifically for the Wii.

To compile:
With Linux: enter "make" in a terminal in the main directory
